## Шаг 1 — Установить Flutter

### На Windows:
1. Скачать Flutter SDK: https://docs.flutter.dev/get-started/install/windows
2. Распаковать в `C:\flutter`
3. Добавить `C:\flutter\bin` в PATH

### На macOS:
```bash
brew install flutter
```
или скачать с https://docs.flutter.dev/get-started/install/macos

### На Linux:
```bash
sudo snap install flutter --classic
```

### Проверить установку:
```bash
flutter doctor
```
Все галочки должны быть зелёными (кроме iOS, если не нужен).

## Шаг 2 — Установить Android Studio

1. Скачать: https://developer.android.com/studio
2. Установить
3. Открыть Android Studio → SDK Manager → установить:
   - Android SDK Platform 33 (или новее)
   - Android SDK Build-Tools
   - Android Emulator
4. Принять лицензии:
```bash
flutter doctor --android-licenses
```

## Шаг 5 — Сборка APK

### Debug APK (для быстрого тестирования):
```bash
flutter build apk --debug
```

### Release APK (для постоянного использования):
```bash
flutter build apk --release
```

APK будет в:
```
build/app/outputs/flutter-apk/app-release.apk
```

---

## Шаг 6 — Установить APK на телефон

### Способ 1 — через USB:
1. На телефоне включить «Параметры разработчика»:
   - Настройки → О телефоне → Нажать 7 раз на «Номер сборки»
2. Включить «Отладка по USB»
3. Подключить телефон через USB
4. Выполнить:
```bash
flutter install
```
или через adb:
```bash
adb install build/app/outputs/flutter-apk/app-release.apk
```

### Способ 2 — через файл:
1. Скопировать APK на телефон (через USB, Bluetooth, Telegram себе)
2. Открыть файловый менеджер на телефоне
3. Найти APK и нажать на него
4. Разрешить установку из неизвестных источников
5. Установить

## Возможности приложения

### Главный экран
- Карточка автомобиля с фото (нажмите чтобы сделать фото)
- Пробег (нажмите для обновления)
- Прогресс-бар до следующего ТО
- Быстрые кнопки перехода

### История обслуживания
- Добавление записей с фото чеков
- Сортировка по дате
- Стоимость каждой работы

### ТО
- 8 позиций с интервалами
- Цветовая индикация: зелёный / оранжевый «скоро» / красный «просрочено»
- Прогресс-бар для каждой позиции

### Запчасти (самый важный раздел)
- OEM номера + аналоги + лучший аналог
- Фото детали и фото коробки
- Статусы: Нужно купить / Заказано / Получено / Установлено
- Поиск по OEM, названию, аналогам
- Фильтр по категориям и статусам
- Вкладки: Все / Купить / Заказано / Получено / Установлено

### Диагностика
- Создание неисправностей с фото
- Статусы: В поиске / Ожидает замены / В работе / Устранено
- 5 неисправностей уже добавлены

### Расходы
- Автоматический подсчёт из всех разделов
- Круговая диаграмма по категориям
- Столбчатая диаграмма по месяцам

### Документы
- Фото (из камеры или галереи)
- PDF файлы
- Срок действия страховки (с предупреждением)
- Возможность поделиться документом

### Напоминания
- По дате и/или пробегу
- Отметить выполненным
- Просроченные выделяются красным

### Резервное копирование
- Экспорт базы данных (отправить на почту, в Telegram, в облако)
- Импорт при смене телефона

## Добавление фото (камера)

В каждом разделе кнопка **«Добавить фото»**:

- Фото хранятся на телефоне, в базе данных — путь к файлу

## Решение проблем

### `flutter doctor` показывает ошибки Android SDK:
```bash
flutter doctor --android-licenses
# Нажать y на все вопросы
```

### Ошибка при сборке «Gradle failed»:
```bash
cd android
./gradlew clean
cd ..
flutter build apk --release
```

### Gradle не скачивается:
- Нужен интернет при первой сборке
- Gradle скачивает ~500 МБ зависимостей

### APK не устанавливается на телефон:
- Включить «Установка из неизвестных источников» для файлового менеджера
- На Android 8+: разрешение даётся отдельно каждому приложению
